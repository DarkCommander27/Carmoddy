# Carmoddy - 1992 Ford Escort EV Conversion

A comprehensive guide for converting a 1992 Ford Escort to an electric vehicle using parts from a Nissan Leaf.

## Overview

This project documents the conversion of a classic 1992 Ford Escort to a fully electric vehicle using the electric motor, battery pack, and related components from a Nissan Leaf. This conversion maintains the original character of the Escort while providing modern electric propulsion.

Before beginning this conversion, ensure you have:

- Mechanical and electrical expertise
- Proper tools and workspace
- Understanding of high-voltage safety procedures
